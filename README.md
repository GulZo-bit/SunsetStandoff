# Sunset Standoff
This was a project made for the DMU game dev socities game jam. We were given a theme of the old wild west and decided to create a game based around duelling. 

Developed in Unity and coded in C#. Sunset Standoff is a local 2 player game where the goal of the game is to be more precise and faster to draw your weapon than your opponent.

The players play on the same keyboard with them having to press their enter key ,P1 uses the main enter key; P2 uses the numbad enter key, faster and more accurately than one another.
By accurate I mean that there are two moving bars that represent accuarcy with the edges having green areas. Whoever gets the bar in the green area first will win.
